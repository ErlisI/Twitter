# Twitter Clone

This is a basic Twitter app to view, compose, favorite, and retweet tweets.

## User Stories

The following **required** functionality is completed:

- [x] User can compose a tweet.
- [x] User can favorite a tweet.
- [x] User can retweet a tweet
